# Репозиторий для работы по курсу "Разработка систем глубокого обучения"
Репозиторий содержит:
1. Файл `stitching_sample.cpp`, который реализует пайплайн SuperPoint + LightGlue для
   сшивки панорамы.
2. Файл `CMakeLists.txt` для сборки приложения.
3. Директорию `models`, в которой лежат модели.
4. Директорию `docker`, в которой лежит файл с докер-образом.
