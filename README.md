# Репозиторий для работы по курсу "Разработка систем глубокого обучения"
Репозиторий содержит:
1. Файл `stitching_sample.cpp`, который реализует пайплайн SuperPoint + LightGlue для
   сшивки панорамы.
2. Файл `CMakeLists.txt` для сборки приложения.
3. Директорию `models`, в которой лежат модели.
4. Директорию `docker`, в которой лежит файл с докер-образом.

## Сборка контейнера
Для сборки докер-контейнера необходимо прописать следующую команду:
```bash
cd docker/ && docker build . -t pano-stitcher
```
Во время сборки будет собираться `OpenCV` и скачиваться `ONNXRuntime`, после чего
соберется само приложение для сшивки панорамы.
В качестве теста корректности будет использоваться попиксельное сравнение панорамы, построенной в контейнере
и построенной заранее.

Если возникли проблемы с `CUDA`, то можно запустить пайплайн на CPU, для этого внутри `Dockerfile` в строке 
с запуском скрипта заменить "cuda" на "cpu".
